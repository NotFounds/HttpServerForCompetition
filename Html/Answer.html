Problem1 FizzBuzz - Answer

問題

「Fizz Buzz」と言われる数字を使ったゲームがあります。このゲームは1から順番に数字を一つずつ数え上げていき、
3で割り切れる場合は「Fizz」、5で割り切れる場合は「Buzz」、両者で割り切れる場合は「FizzBuzz」と言います。
あなたは100まで数え上げなければなりません。1から順に100までFizzBuzzを表示するプログラムを作成してください。

入力

入力はありません。

出力

1から100まで順番に数を表示します。
その数が3で割り切れるなら「Fizz」
5で割り切れるなら「Buzz」
両方で割り切れるなら「FizzBuzz」と表示してください。

制約条件

改行を入れてください。
FizzBuzzと表示するときは、FizzとBuzzの間にスペースは入れないでください。
すべて半角英数で出力するようにしてください。
実行時間は1000ms以内です。


出力例

1
2
Fizz
4
Buzz
...

解法1

愚直に1から100まで標準出力を用いて出力する
もちろんこれでもOKですが...
もし、1000まで表示のような問題だったら大変ですので、次の解法

解法2

For文もしくはWhile文等のループを用いて、1から100まで繰り返す
ループの中でIf文を用いて3で割り切れたら「Fizz」、5で割り切れたら「Buzz」
どちらでも割り切れたら「FizzBuzz」、どちらでも割り切れなかったら数字を出力


解法2のソースコード

C言語 
#include <stdio.h>
int main()
{
	int i;
	for (i = 1; i <= 100; i++)
	{
		 if (i % 15 == 0)
		 {
			printf("FizzBuzz\n");
		 }
		 else if (i % 3 == 0)
		 {
			printf("Fizz\n");
		 }
		 else if (i % 5 == 0)
		 {
			printf("Buzz\n");
		 }
		 else
		 {
			printf("%d\n", i);
		 }
	}
	return 0;
}
